#  N101_Json字段

## 1.说明
* 对上服务，在资源允许的情况下推荐使用Json。方便平台对接。
* 对于成套产品，需考虑Json字段包括哪些内容。
* 该文档经针对N101**前期开发阶段**的上报字段做字段说明。
* Json字段可参考**二进制流的相关字段报文**。建议字节流和json的字段内容，协议做统一。
