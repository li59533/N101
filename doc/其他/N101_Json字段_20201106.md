#  N101_Json字段
"cat":**"nb_n"**,		  // 设备类别
"profile":**0**,	  // 协议版本


----------------------------
## 1.说明
*  基于**“json通讯协议说明V0.0.1”**开发

## 2.详细协议
**json格式**

```json
{
	"sn": "88888888", // SN
	"cat":"nb_n",		  // 设备类别
	"profile":0,	  // 协议版本
	"rssi":25,
	"ack":0,
	"cmd": xxxx,		  // 命令字
	"payload": {	  // 数据
        "xxx": xxxx
	}
}
```
### 2.1自定义cmd
```c
typedef enum
{
    128 ,//上报X轴特征值
    129 ,//上报y轴特征值
    130 ,//上报z轴特征值
    131 ,//上报x轴波形
    132 ,//上报x轴波形
    133 ,//上报x轴波形
    134 ,//配置请求
    135 ,//配置应答
    136 ,//历史数据请求
    137 ,//历史数据应答
    138 ,//报警上报
    
}selfcmd_e;
```
